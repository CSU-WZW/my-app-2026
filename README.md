# 时光日程 (Planner) 🕒

这是一个基于 HTML/CSS/JS 构建的极简、高效的日程管理应用程序。它旨在帮助用户轻松规划每一天，提升工作和生活效率。

## 🌐 在线演示

您可以直接访问部署在 GitHub Pages 上的应用：
[https://o226t025-hash.github.io/my-app-2026/](https://o226t025-hash.github.io/my-app-2026/)

## ✨ 功能特性

- **日程管理**：快速创建、查看和管理您的每日任务。
- **分类筛选**：支持对日程进行分类，帮助您更好地组织任务。
- **今日聚焦**：突出显示当天的核心任务，保持专注。
- **响应式设计**：完美适配手机、平板和桌面设备。
- **暗黑模式**：支持一键切换深色主题，保护视力。
- **美观界面**：基于 Tailwind CSS 构建，拥有现代、简洁的设计风格。

## 🚀 如何使用

1. **添加日程**：点击页面右上角的“创建日程”按钮，填写任务内容并保存。
2. **切换视图**：通过左侧面板切换不同的筛选分类。
3. **切换模式**：点击顶部的月亮/太阳图标，在亮色和暗色模式间自由切换。
4. **管理状态**：标记已完成的任务（即将推出更多交互功能）。

## 🛠️ 技术栈

- **HTML5**：页面结构。
- **Tailwind CSS**：响应式样式与布局。
- **FontAwesome**：图标支持。
- **JavaScript (ES6+)**：交互逻辑与数据处理。
